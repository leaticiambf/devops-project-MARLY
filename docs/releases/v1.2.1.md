# Mavigo v1.2.1 Release Notes

Release date: 2026-06-15

## Highlights

- Displays the live map directly on the Journey page even before a journey is planned.
- Keeps the map available as a network exploration surface while the planner remains ready for input.
- Polishes the responsive frontend experience with a mobile-first quick access navigation.
- Replaces the legacy `MV` mark with the Mavigo logo in the application shell and authentication screens.

## Frontend

- Updated the Journey workspace desktop layout so the live map is always visible beside the planner.
- Added a mobile/tablet live map section when no journey is active.
- Preserved the active journey map behavior for routed segments, stops, tasks, tourist markers, rerouting, and live location.
- Improved the authentication and restoring-access screens with consistent logo presentation.
- Added mobile bottom navigation and quick access cards for the main features.

## Documentation

- Added release notes for `v1.2.1`.
- Updated the changelog with the patch release scope.

## Compatibility Notes

- This patch builds on `v1.2.0` and keeps the tourist mode and map feature set intact.
- Map rendering still depends on the frontend Mapbox token configuration.
