# Mavigo v1.2.1 Release Notes

Release date: 2026-06-15

## Highlights

- Keeps the live map available without showing a demo itinerary when no journey is planned.
- Polishes the responsive frontend experience with a more compact mobile journey flow.
- Adds a dedicated mobile-friendly Comfort Presets page for saved route preferences.
- Improves Eco Score readability on phone screens with smaller cards and denser badge layout.
- Replaces the legacy `MV` mark with the Mavigo logo in the application shell and authentication screens.

## Frontend

- Updated the Journey workspace desktop layout so the live map is visible beside the planner.
- Prevented demo route segments from rendering on Journey and Map pages when no route is active.
- Preserved the active journey map behavior for routed segments, stops, tasks, tourist markers, rerouting, and live location.
- Reworked mobile active-journey mode so the planner is hidden while the user follows a trip.
- Switched mobile trip options from a horizontal strip to a vertical itinerary layout.
- Added `/comfort-presets` with list, create, edit, and delete flows for named comfort settings.
- Simplified the mobile account menu to focus on account, home address, logout, and a link to Comfort Presets.
- Compacted Eco Score metrics, badge cards, and release-safe toast placement for phone viewports.
- Improved the authentication and restoring-access screens with consistent logo presentation.
- Added mobile bottom navigation for Journey, Explore, Tasks, Eco Score, and Comfort Presets.

## Documentation

- Updated release notes for `v1.2.1`.
- Updated the changelog with the final patch release scope.
- Updated frontend architecture notes for the new Comfort Presets route/module.

## Compatibility Notes

- This patch builds on `v1.2.0` and keeps the tourist mode and map feature set intact.
- Map rendering still depends on the frontend Mapbox token configuration.
