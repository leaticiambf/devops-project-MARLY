# Mavigo v1.2.0 Release Notes

Release date: 2026-05-04

Tag target: `52a99059d8714b12493f3c6cb18a94146585f4b4`

## Highlights

- Added the interactive transport map feature.
- Added a dedicated transport map page in the Next.js frontend.
- Displayed included journey tasks as routed waypoints on the map and in the route panel.
- Added reverse geocoding utilities for map interactions.
- Improved map availability error handling and user feedback.

## Backend

- Updated journey response data used by the map-enabled frontend.
- Adjusted journey optimization behavior for map-aware route presentation.

## Frontend

- Added `TransportMap`, map legend, map utilities, and map workspace components.
- Added Mapbox integration support for the interactive map experience.
- Extended the journey workspace to render richer route and waypoint map states.
- Added map-related page coverage to frontend tests.

## Documentation

- Updated README metadata and badges.
- Documents that v1.2.0 includes all v1.1.0 tourist mode changes because `23f8b6f` is an ancestor of `52a9905`.

## Compatibility Notes

- This release builds on v1.1.0 and includes the tourist mode feature.
- Map behavior depends on the frontend map configuration and available browser/runtime support.
